1. Back Story:
   
    Spare My Life.
        Glen Dashner is an Outlaw, living out on the fringe of civilized space. One day he was accused of a crime, he claimes he didnt commit, in a sector of space he's never been.
        But now as he stands trial and put to the gallows, Its up to YOU the player to determine his innocence. 

3. How to get Started:

Link To The Game:

     A.  START GAME:
 
    Click the start game button to start the game.
    A theme related word from a predetermined list will randomly generate as the word to be guessed.

     B. RESET GAME: 
 
     Click the reset game button
     to reset the game, to save Glen's life or your just not a fan of losing.

    C. SHOW HINT:

    Show Hint displays a hint for the predetermined word that was selected at random at the start of the game.

     D. Original Planning:

    Game Name: Spaceman “hangman”
    
        Pseudocode:
    
    Define Variables
    
        If player selects the correct letter “place it”
        
        If player selects the incorrect letter display a “limb”

    Initialize Game
    
        Set starting gallows empty
        
        Display these values on the webpage.

    Create Event Listeners
    
        Check for letter click

    Define Game Loop Function
    
        Check for correct letter selection
        
        Check for incorrect letter selection

    Check for Game Over
    
        If player chooses too many incorrect letters and spaceman is hung
        
        Stop the loop.

     Update Display Function
 
        Refresh stats on screen each loop or button click.
        
        change spaceman’s additional limbs per incorrect selection
